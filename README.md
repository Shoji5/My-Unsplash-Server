## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

2. Clone the repo
   ```sh
   git clone https://github.com/Shoji5/My-Unsplash-Server.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Create your own `.env` file
   ```sh
   PORT = YOUR_PORT
   MONGOOSE_URL = YOUR_MONGOOSE_URL
   ```

## Usage
